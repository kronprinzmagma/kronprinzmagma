## Selected work

Four production-grade projects. Each is a public, sanitised mirror of a private working repo, kept in sync after every release. The business problem comes first in every README — the engineering proof (architecture, tests, evals, cost-tracking) is one click away inside each repo.

- **[agentic-cv-tailoring](https://github.com/kronprinzmagma/agentic-cv-tailoring)** — Reframes a documented CV for a specific role without inventing anything: a dedicated factcheck agent vetoes any claim that can't be traced to a source document. 8-agent pipeline, cross-provider review (Anthropic + OpenAI), deterministic quality gates. _Public version of `cv-tailor`._
- **[ai-listing-assistant](https://github.com/kronprinzmagma/ai-listing-assistant)** — Photo → complete bilingual (DE/FR) Ricardo.ch listing in under two minutes. 5-agent pipeline, a published MCP server, and an LLM-as-judge eval suite. _Public version of `verkaufshilfe-via-foto`._
- **[pdf-document-organizer](https://github.com/kronprinzmagma/pdf-document-organizer)** — Turns a pile of `scan0042.pdf` into clean, searchable filenames — without ever sending personal data to the cloud. PII is redacted in memory before any cloud call; can run fully local. _Public version of `dok-namer`._
- **[ki-news-aggregator](https://github.com/kronprinzmagma/ki-news-aggregator)** — A daily AI briefing for product people who build, not just spec. 15 sources, Claude Haiku scoring + Sonnet writing, published as GitHub Issues with a live archive. [Live archive](https://kronprinzmagma.github.io/ki-news-aggregator/).
